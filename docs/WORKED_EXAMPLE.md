## The point

Two passes with nearly identical downstream value (ΔP): SKM v1 scores the
pressured, late, difficult one **1.2×** higher. That gap — invisible
to pure outcome metrics — is what SKM is for.

Note the v1.5 adjusted layer *narrows or flips* this particular comparison: the routine pass was progressive for the passer's position and fed a chain ending in a shot, so its position and sequence weights lift it. Both effects are visible in the decomposition above — that transparency is the design goal, and it is exactly the kind of interaction a single black-box number would hide.


*Caveats: D/C/R are v1 models on event data only (no tracking); "line-
breaking" cannot be verified without defender positions, so selection used
forward distance + pressure as proxies.*
